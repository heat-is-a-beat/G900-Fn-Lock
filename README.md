# G900-Fn-Lock
Casper Excalibur G900 için fn lock AHK scripti. Diğer notebook modellerinde denenmedi. AutoHotkey v2 gerektirir. Cihazın HotkeyOSD driverinın sadece parlaklık için sağlamadığı ui elementini içinde barındır (Cihazınızın driverlerini indirmeniz tavsiye edilir.). Görev zamanlayıcı ile yönetici yetkileriyle otomatik başlatma ayarlanması tavsiye edilir.
